# KiCad footprint library
Each footprint has a separate library file to simplify tracking of changes.

## List of components
```bash
├── Connector
│   └── PMOD_SpecB.pretty
├── Inductor    
│   └── LPS3015.pretty
├── Mechanical
│   └── Hole.pretty
├── Regulator_Switching
│   └── TPS63031.pretty
├── Sensor_Motion
│   ├── BMG250.pretty
│   ├── LIS3DH.pretty
│   └── LSM6DS33.pretty
```
